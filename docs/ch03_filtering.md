## 3.2 조건식과 연산자

### 3.2.1 비교 연산자  
- 두 값을 비교
- WHERE절과 함께 사용
  
사용할 비교 연산자 목록:
- `=`  : 같다  
- `!=` : 같지 않다  
- `>`  : 크다 (초과)  
- `>=` : 크거나 같다 (이상)  
- `<`  : 작다 (미만)  
- `<=` : 작거나 같다 (이하)  

### 3.2.2 논리 연산자  
- 조건을 조합하거나 반전

사용할 논리 연산자 목록:
- `AND` : 두 조건을 모두 만족하는 대상(교집)  
- `OR`  : 두 조건을 모두 만족하는 것(합집합)  
- `NOT` : 조건을 만족하지 않는 것(여집합)
